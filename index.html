How to Find Your Player ID

▶
  1. Open the Season 10 Leaderboard
  2. Find and click your player name
  3. In the URL, copy the string of numbers after /player/...
  4. Paste them into the search box

How to Add Overlays to OBS

▶
  1. Open an overlay and copy its full URL
  2. In OBS, select Sources → + → Browser
  3. Select "Create New"
  4. Paste URL and remove default CSS
