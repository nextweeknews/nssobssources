How to Find Your Player ID

▶
  1. Open leaderboard
  2. Select your player name
  3. Copy the numbers after /player/
  4. Paste above!

How to Add Overlays to OBS

▶
  1. Copy full overlay URL
  2. OBS → Sources → + → Browser
  3. Select “Create New”
  4. Paste URL, set dimensions

Recommended Dimensions

▶
  • Simple Elo: 600×240
  • Last Match: 480×400
  • Rank Position: 450×600
  • Leaderboard: 450×650

Troubleshooting

▶
  • If overlay freezes:
  • Disable OBS Browser Source Hardware Acceleration
  • Restart OBS afterwards
